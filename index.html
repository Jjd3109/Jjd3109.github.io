Skills

저는 2022.03 ~ 2022.10 까지 약 7개월간 KH정보교육원에서 스프링기반 자바강의를 들으면서 JAVA를 기본으로 ORACLE, Spring Framework, MYBATIS, HTML, CSS, JavaScript를 공부하였습니다. 현재도 Spring Framework를 이용한 공부를 진행하고 있으며, 쇼핑몰같은 홈페이지는 시간이 걸리지만 혼자서 개발할 수 있습니다. 웹개발을 하다보니 wordpress도 관심이 있어 같이 공부하고 있고, 새로운 프로젝트를 통해 개인적으로 생각해왔던 서비스를 구현해보려고 노력하고 있습니다.

HTML 50%

웹서비스 화면을 직접 구현해보면서 JSTL을 사용하여 공부를 하였습니다.

CSS 50%

부트스트랩등을 이용하여 CSS를 깔끔하게 만들 수 있고 간단한 부분을 만질 수 있습니다.

JavaScript 60%

JavaScript의 기본문법, jquery를 이용해서 웹프론트에서 구현과, ajax를 통한 여러가지를 구현할 수 있습니다.

JAVA 70%

기본적인 문법과, 알고리즘 공부와 자바스프링을 주로 공부하였습니다.

ORACLE 50%

oracle sql 기본문법등을 사용할 수 있고, 검색기능과 여러가지 기능을 sql로 구현해보려고 했습니다.

Portfolio

팀원들과 처음으로 함께한 프로젝트로써 혼자가 아니라 팀원들끼리 하면서 협업에 대한 것을 배우고, 그동안 학원에서 배웠던 ORACLE, Spring Framework, MYBATIS, HTML, CSS, JavaScript 등을 다 써보면서 조각조각 배웠던 것들이 합쳐지면서 여러가지를 활용하면서 계속 만들었던 결과 실력이 많이 늘었던 계기가 되었고 더더욱 프로그래밍에 대해 흥미를 느끼게 된 프로젝트였습니다.

주요기능

1. 회원가입

로그인, 비밀번호 찾기, api소셜로그인, 판매자 소비자 분리 회원가입후 유형별 접근 제한, 이메일인증

2. 마이페이지

개인정보 수정/삭제, 관심상품, 구매내역

3. 상품거래

상품 및 리뷰조회 등록 / 수정 / 삭제, 관심상품 조회 / 등록 / 삭제 / 쿠폰 및 포인트 사용가능한 결제 연동, 상품 리스트 페이징, 조회수 및 등록일 조회

4. 관리자 페이지

회원정보 및 거래내역 조회
